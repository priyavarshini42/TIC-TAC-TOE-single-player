# 🎮 TIC-TAC-TOE – X or O

A modern, responsive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript** featuring an intelligent AI opponent powered by the **Minimax Algorithm**.

## ✨ Features

* 🎨 Modern Glassmorphism UI
* 💜 Lavender, Pink, Violet & Indigo Theme
* 📱 Fully Responsive Design
* 🤖 Unbeatable AI using Minimax Algorithm
* 🎯 Choose to Play as X or O
* 🏆 Win, Lose, and Draw Detection
* 🎉 Animated Victory Celebration
* 📊 Persistent Scoreboard using Local Storage
* 🔄 Reset Board & Restart Match Options
* 🔙 Back to Symbol Selection Screen
* ✨ Smooth Animations and Hover Effects
* 🌈 Animated Gradient Background
* 🎊 Confetti Celebration Effects

## 🚀 How to Run

### Method 1: Open Directly

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.

The game will launch in your browser.

### Method 2: Using VS Code

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

---

## 🎮 How to Play

1. Launch the game.
2. Choose your symbol:

   * X
   * O
3. Take turns placing symbols on the board.
4. The computer immediately responds with its move.
5. Get three symbols in a row to win.
6. If all cells are filled without a winner, the game ends in a draw.

---

## 🤖 AI Implementation

The computer opponent uses the **Minimax Algorithm**, a decision-making algorithm commonly used in game theory.

### AI Capabilities

* Always chooses the best possible move.
* Blocks player winning opportunities.
* Finds winning combinations.
* Prioritizes optimal board positions.
* Plays nearly perfectly.

Because of Minimax, the AI is extremely difficult to defeat.
